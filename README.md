# GliderFlightCore

Открытая система для управления полетными режимами авиамодели планера F1C на базе бортового модуля ESP8266 и мобильного приложения на Flutter.

Проект предназначен для замены механических таймеров, предоставляя гибкость в настройке полетной программы и возможность сбора и анализа телеметрии.

## Основные возможности

-   **Профили планеров:** Создание и хранение отдельных профилей для каждой авиамодели.
-   **Редактор полетных программ:** Гибкая настройка последовательности срабатывания сервопривода по временной шкале.
-   **Беспроводная настройка:** Загрузка полетной программы на бортовой модуль по Wi-Fi.
-   **Автономный полет:** После загрузки программы модуль работает полностью автономно, включая отключение Wi-Fi для экономии энергии.
-   **Запись телеметрии:** Сбор данных о высоте полета с барометрического датчика.
-   **Анализ полетов:** Скачивание логов полетов на телефон для просмотра статистики и графиков.

## Технологии

-   **Бортовой модуль (Hardware):**
    -   Микроконтроллер: ESP8266 (Wemos D1 Mini)
    -   Датчики: Барометр (BMP/BME280)
    -   Исполнительные механизмы: Сервопривод
-   **Прошивка (Firmware):**
    -   Язык: C++
    -   Среда: PlatformIO / Arduino Core
-   **Мобильное приложение (Mobile App):**
    -   Фреймворк: Flutter
    -   Язык: Dart
-   **Протокол обмена:**
    -   Связь: Wi-Fi (точка доступа / клиент)
    -   API: RESTful API (HTTP/JSON)

## Структура проекта

-   **/firmware**: Исходный код прошивки для ESP8266.
-   **/mobile_app**: Исходный код мобильного приложения на Flutter.
